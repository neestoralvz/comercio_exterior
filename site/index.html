    Documentación de Comercio Exterior¶

    ¡Bienvenido a la Documentación de Comercio Exterior! Este repositorio contiene toda la información necesaria para llevar a cabo el proceso de exportación de mercancías peligrosas —en particular, ácido sulfúrico— desde México hacia Guatemala, siguiendo las mejores prácticas de comercio internacional.

    ¿Qué incluye esta documentación?¶

    1. Precios y Pagos

    2. Contiene los lineamientos para la fijación de precios y los términos de pago sugeridos para operaciones de exportación.

    3. Normativas y Regulaciones

    4. Reúne la legislación y requisitos obligatorios en México y Guatemala para manejar y transportar sustancias químicas peligrosas.

    5. Incoterms

    6. Explica los términos de comercio internacional más adecuados para la transacción, especificando responsabilidades y costos de cada parte.

    7. Requisitos Legales y Aduaneros

    8. Detalla la documentación necesaria para exportar e importar, así como información de agentes aduanales y permisos especiales.

    9. Entrega Suchiate

    10. Incluye consideraciones de logística para transporte en pipa de acero al carbón, así como protocolos de seguridad para el traslado.

    11. Formalidades Frontera

    12. Describe los pasos específicos para la salida de mercancías desde México y la entrada a Guatemala, con lineamientos aduaneros y sanitarios.
