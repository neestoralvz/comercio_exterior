 Documentación de Comercio Exterior
=========================================

información y procedimientos para la exportación de ácido sulfúrico desde México
hacia Guatemala, así como requisitos, normativas y mejores prácticas.

Generación de la Documentación
==============================

Este proyecto usa Sphinx, el cual convierte tus archivos reStructuredText (o 
Markdown mediante extensiones) en un sitio HTML navegable. Para generar la 
documentación, ejecuta:

.. code-block:: bash

   make html

en la carpeta raíz (o en la carpeta donde resida tu Makefile).
