Resumen#

Este documento establece el marco integral para la gestión de riesgos en el transporte de Ácido Sulfúrico entre México y Guatemala, siguiendo las normativas NOM-002-SCT/2011, RTCA 71.03.37:07 y estándares internacionales ISO 31000:2018.
