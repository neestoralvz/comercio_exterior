Introducción
===========

Esta sección detalla los requisitos legales y aduaneros necesarios para la exportación de |product| desde |origin_country| hacia |destination_country|.

Propósito
---------

Establecer los requisitos y procedimientos legales y aduaneros necesarios para asegurar una exportación conforme a las normativas de ambos países.
